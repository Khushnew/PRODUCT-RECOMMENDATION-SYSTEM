# PRODUCT-RECOMMENDATION-SYSTEM

## Overview
A simple **Product Recommendation System** built in Python that generates product suggestions based on data and outputs reports in CSV and PDF formats.  

---

## Project Structure
/Recom_proj/
├─ proj.sql # MySQL database setup
├─ project.py # Main Python code
├─ recommendation_report.csv # Sample CSV output
├─ recommendation_report.pdf # Sample PDF output
└─ README.md # Project documentation


---

## Prerequisites
Make sure the following are installed:

- **Python 3.x**  
- **Tkinter** (for GUI, optional if not using GUI)  
  ```bash
  sudo apt-get install python3-tk
MySQL Connector

pip install mysql-connector-python
Pandas

pip install pandas
Matplotlib (if plotting is used)

pip install matplotlib
FPDF (for PDF reports)

pip install fpdf
Usage
Set up the MySQL database using proj.sql.

Run the main Python script:

python project.py
Check the generated output:

recommendation_report.csv – CSV report

recommendation_report.pdf – PDF report
